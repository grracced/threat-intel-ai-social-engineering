# AI Social Engineering Threat Intelligence Report

## Overview

This project is a Cyber Threat Intelligence (CTI) report focused on AI-powered social engineering attacks and how threat actors use artificial intelligence to manipulate victims through deepfake technology, voice cloning, phishing, and impersonation attacks.

The report was developed as part of:

**The Purpose Protocol — Cohort 1 (2026)**  
Project Track: **AI Cyber Threat Intelligence (AI CTI)**

---

## Project Objectives

The goal of this project is to:

- Understand how AI enhances modern social engineering attacks
- Analyze real-world deepfake scam incidents
- Map attacker behavior using the MITRE ATT&CK framework
- Identify detection methods and organizational risks
- Recommend practical cybersecurity defenses

---

## Key Findings

- AI is making social engineering attacks more realistic and scalable
- Deepfake voice and video technology can impersonate trusted individuals
- Human emotions such as urgency, fear, and authority are major attack vectors
- Financial departments and organizations using digital communication are high-risk targets
- Employee awareness and verification procedures remain critical defenses
